# README

This project is a reimagining of the _Reincarnate_ spell from Dungeons & Dragons 5th Edition.

See https://mpanighetti.github.io/dnd5e-updated-reincarnate-tables for a published browsable webpage. This repository is a GitHub-managed version of the published page. Feel free to contribute to this project by submitting issues or pull requests.
